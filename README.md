# 📚 ChatVault · 全局聊天档案管理器

> *让你像翻书架一样，随心翻阅自己所有的聊天记录！*

**当前版本：v0.3.24**

---

## ✨ 一、它能给你什么

**1. 🌐 全局视野，告别繁琐**
不管你在酒馆开始界面、还是已经进了某个角色卡，魔法棒里点开 [聊天档案]，所有角色、所有聊天文件一目了然。再也不用一张张点进角色卡找记录了！
同时你也可以快捷管理 导出 和 删除。

**2. 🚀 随心跳转**
看到喜欢的旧聊天，一键「**继续**」直接跳进去接着聊。已经在用的那条会标「**使用中**」，绝不会弄混。

**3. 📖 沉浸式阅读 —— 真·像看小说**
重头戏！任何一条聊天，点「**阅读模式**」就能从第一楼平铺翻到最后一楼。
自带分页器、字号调节、首行缩进。**不用导出、不用解析 JSONL、不用切换页面**，原汁原味直接读！依赖摘取规则，让正文清清爽爽。

**4. ⚡ 更快捷的新建聊天**
搜索角色 → 点 + 一键新建，省略中间步骤。

**5. 🧹 更干净的导出**
* **txt**：按你设定的「摘取规则」清洗后导出。
* **jsonl**：原始格式完整保留。

## 🗂️ 二、四大模块速览

### 📑 主面板 (四个 Tab)

* 🕒 **最近**：全角色聊天按「最后消息时间」排序，50 条/页。
* 👥 **按角色**：按角色分组折叠，找谁点谁。
* ⭐ **收藏**：你点过星的聊天集合页。
* 🔷 **当前角色**：点击可查看当前打开的角色卡所有聊天记录。

> *每条卡片显示：头像 · 角色名 · 自定义标题 · 原始文件名 · 消息数/大小/相对时间 · 你打的标签 · 最后一条消息预览（懒加载，不卡顿）*

### 👓 阅读模式

点卡片底部进入。书页式排版、头像左右对齐（user 也能绑定 persona 头像）、行内 **粗体** 和 *斜体* Markdown 自动渲染。滚动时浮动按钮自动隐藏让你专心看，分页器翻页，每页 30 层。

### ✂️ 摘取规则

告诉 ChatVault：哪些是废话该剥掉，哪些才是正文。**阅读模式与 txt 导出共用一套，改一次到处生效。**

### ✏️ 收藏 & 编辑

* ★ **一键收藏**
* ✎ **编辑**：自定义标题（覆盖原文件名显示，但不改服务器）、打标签、真正重命名服务器上的文件。

### 🔍 全局搜索

顶部搜索框支持搜索：**角色名 / 自定义标题 / 文件名 / 标签** —— 任何一个匹配都会高亮命中！

## ⚠️ 三、关于「快照」（请认真看一眼）

> **ChatVault 用了索引快照机制让面板"瞬开"：**
> ✅ **快照存的是清单**（哪个角色有哪些聊天文件、文件名、大小、时间）
> 🟡 **每次刷新浏览器会失去快照**，再打开 ChatVault 会重新加载最新索引。
> ❌ **快照不存聊天正文**，更不会备份你的聊天。

🚨 **核心警告：**

* 面板里看到的卡片不是副本，是你服务器上真实文件的**索引**。
* 点 🗑 删除 按钮 = **真实删除**服务器上的 JSONL 文件，删了就彻底没了！
* 新建聊天/新楼层，ChatVault 不会自动感知。点标题栏的 ⟳ 刷新键 即可加载最新状态。
* *快照旁的小圆点提示：[聊天档案]右边的圆点出现 = 当前显示的是缓存数据；点刷新后小圆点消失 = 已是最新索引。*

## 🛠️ 四、操作指南

### ⚙️ 摘取规则怎么用

任何一张卡片的折叠区里都有「**摘取规则**」的入口，点开是弹窗。规则分两段：

**1️⃣ 剥离（Strip）—— 该删的删掉**

* `<thinking>` / `<think>` —— AI 思考链
* `<!--HTML 注释-->`
* `<recall>` / `<supplement>` —— 召回/补充块
* `<PascalCase/>` —— 自闭合占位标签（如 `<StatusPlaceHolder/>`）
* `# Markdown 标题行` —— 整行 `### xxx` 这种
* 自定义标签对（自己填 `<开标签>` / `</闭标签>`）

**2️⃣ 提取（Extract）—— 只保留指定标签内的内容**

* `<content>` / `<reply>` / `<本轮用户输入>` —— 常见正文容器
* 自定义标签对

> 👤 **user 单独规则**：勾上后，user 消息会走另一套（默认开 `<recall>` `<supplement>` 剥离 + `<本轮用户输入>` 提取，符合大多数预设的写法）。改完即时保存，下次开阅读模式/导出**立即生效**。

**你只要设置了，就是全局套用。**

### 📖 阅读模式设置面板 (齿轮图标)

* **字号**：11–22 px 滑块，正文跟着变。
* **首行缩进**：段落首行缩进 2 字符，更像小说排版。需要开启按钮。
* **分页器模式**：可选「常驻底部」始终显示，或「下滑隐藏 / 上滑出现」沉浸阅读时不打扰。
* **绑定 user 头像**：选一个 persona 头像贴上你的发言。再次点选同一个 = 解绑。（读不到 persona 列表时可手动输入文件名）。

> *注：本来想直接读取 user 用户名和头像，但头像似乎无法从技术上实现，绑定是一个补救办法。链接你酒馆本来的文件，不会拷贝副本。*

### 🎨 主题切换

在 插件设置（齿轮 → ChatVault → 主题）三选一：
🌙 **夜间 Dark** | ☀️ **白底 Light** | ☕ **咖啡 Coffee**

## 🛡️ 设计原则

* 🛑 **零后台运行**：关闭面板 = 关闭一切监听器、计时器、观察者。
* 🛑 **零第三方网络**：所有图标内联、无外链字体、无遥测、无统计。
* 🛑 **零数据外发**：聊天内容 100% 留在你自己的酒馆服务器。
* 🍃 **轻量克制**：只在你主动操作时工作。
